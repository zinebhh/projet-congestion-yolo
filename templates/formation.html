Introduction

Cette documentation présente la formation à l'analyse du trafic routier basée sur l'utilisation de technologies modernes comme YOLO (You Only Look Once), DeepSORT, et Streamlit. Le but est de créer une application complète de détection de congestion, comptage de véhicules, estimation de vitesse et visualisation des données.

Le programme est conçu pour les étudiants en informatique souhaitant apprendre à combiner l’intelligence artificielle, la vision par ordinateur et le développement web pour des projets concrets. Cette formation est aussi adaptée pour les projets de fin d’études (PFE).

Support

Cette documentation est organisée en modules clairs pour faciliter l’apprentissage et l’intégration des différentes parties du projet.

Pour toute question ou suggestion, veuillez utiliser l’espace de feedback intégré à l’application ou soumettre une issue sur le dépôt GitHub du projet.

Le projet utilise plusieurs bibliothèques open-source. Si vous rencontrez un bug ou souhaitez proposer une amélioration, merci de soumettre une issue.

Note

  • Ce projet repose sur YOLOv5/v8, DeepSORT, Flask et Streamlit. Pensez à vérifier la compatibilité de vos versions locales avec les dépendances du projet.
  • Certaines bibliothèques tierces sont utilisées pour l’interface graphique et l’analyse de données. Leurs liens sont listés dans la section "Outils Tiers".
  • Vous êtes libre de modifier le projet pour vos besoins pédagogiques ou professionnels, mais mentionnez les auteurs originaux si vous le redistribuez.
  • Aucun support officiel n’est offert sur les bibliothèques tierces. Référez-vous à leur documentation respective.
